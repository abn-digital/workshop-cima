Sección 04

Gemini Enterprise

La plataforma de Google Cloud para construir agentes corporativos con datos reales de tu empresa, controlados y seguros.

Agent Builder

Construí agentes sin código

Agent Builder permite crear agentes conversacionales que conectan con tus datos corporativos: documentos, bases de datos y APIs de terceros.

Ideal para: asistentes de RH, bots de soporte interno, agentes de consulta de datos financieros y más.
Seguridad Enterprise

Control total de los datos

Los datos nunca salen del entorno corporativo. Gemini Enterprise opera dentro de los controles de seguridad de Google Cloud con VPC Service Controls y cifrado CMEK.

Las respuestas del modelo se anclan en los datos reales de la empresa usando RAG (Retrieval-Augmented Generation).
Sección 05

Coding Agents — El siguiente nivel

Más allá de chatear con IA: operar directamente desde agentes que escriben, ejecutan y depuran código. El nivel más profundo del workshop.

Antigravity

Google

El SDK de Google para diseñar, implementar y orquestar agentes autónomos de IA. Permite crear sistemas multi-agente con capacidad de razonamiento avanzado.

Google Antigravity

Claude Code

Anthropic

Agente de terminal de Anthropic que entiende tu codebase completo, ejecuta comandos y hace cambios multi-archivo de forma autónoma.

Claude Code

Cursor

Cursor Inc.

IDE con IA integrada: autocompletado inteligente, chat con contexto del proyecto y agentes que implementan features completos.

cursor.com

¿Por qué es el siguiente nivel?

Mientras las secciones anteriores muestran cómo conversar con IA, acá vemos cómo operar con IA: agentes que leen tu código, entienden tu arquitectura, ejecutan tests y despliegan cambios. Es la frontera actual de productividad con IA.

Sección 06

Independencia total y personalización

Cómo salir de una caja cerrada y ser independiente de manera completa. La convergencia entre el chat propio y la IA de código abierto.

Hike Chat

Ecosistema Hike

Tu propio entorno de conversación e integración de IA. Diseñado para que las personas y organizaciones salgan de una caja cerrada de proveedores únicos, logrando ser independientes de manera completa.

chat.hikethecloud.com

OpenClaw

Asistente de IA Personal

La punta del iceberg: un asistente personal de IA de código abierto que corre en tus propios dispositivos. Se conecta a tus canales cotidianos (WhatsApp, Slack, Telegram, etc.), dándote control absoluto de tus datos y flujos.

OpenClaw en GitHub

Comparativa: OpenClaw vs. Manus AI vs. Hermes Agent

La competencia actual en asistentes y agentes autónomos se resume en el balance entre privacidad, control técnico y comodidad de uso.

Linux de los Agentes

OpenClaw

Código Abierto · Ejecución Local

  • Control local y privacidad absoluta
  • Conexión a WhatsApp, Telegram, Slack
  • Más de 3,000 skills (ClawHub)
  • Configuración técnica (Node.js/CLI)
Apple de los Agentes

Manus AI

Código Cerrado · En la Nube

  • Instalador visual y cero fricción
  • Automatización autónoma "My Computer"
  • Integración fluida en Windows/macOS
  • Suscripción costosa ($39 - $199/mes)
Alternativa Científica

Hermes Agent

Código Abierto · Local / Servidor

  • Sistema robusto de memoria a largo plazo
  • Manejo avanzado de colisiones de herramientas
  • Optimización fina de Nous Research
  • Requiere entorno Python y desarrollo
Característica OpenClaw Manus AI Hermes Agent
Filosofía Hazlo tú mismo (DIY) y control local absoluto. Producto comercial gestionado y pulido. Infraestructura técnica para memoria y herramientas.
Tipo de Código Código Abierto (Open Source) Código Cerrado (Propiedad de Meta) Código Abierto (Open Source)
Ejecución Local-first (datos seguros en tu entorno) En la nube (Cloud-hosted) Local / Servidor (Runtime en Python)
Configuración Compleja (Terminal, Node.js) Cero fricción (Instalador visual) Técnica (Entorno de desarrollo Python)
Costo Gratis (Pagas consumo de API propia) Suscripción ($39 - $199 USD/mes) Gratis (Open Source)
Ecosistema Más de 3,000 habilidades en ClawHub. Capacidades avanzadas nativas de Meta. Registro centralizado de herramientas gobernadas.
¿Cómo elegir? OpenClaw es el "Linux" de los agentes (máxima privacidad y flexibilidad a cambio de configuración técnica); Manus es el "Apple" de los agentes (máxima comodidad pagando una tarifa y delegando el procesamiento de datos a terceros); y Hermes es el runtime ideal para científicos y desarrolladores que necesitan una gestión impecable de memoria a largo plazo.

¿Por qué es clave esta sección?

Mostramos cómo no depender de un único ecosistema corporativo cerrado. Al implementar soluciones abiertas y de control propio como OpenClaw o tu propia interfaz personalizada en Hike Chat, sos el dueño completo de tus integraciones, tus modelos y tus datos.
